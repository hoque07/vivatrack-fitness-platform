# VivaTrack Fitness Platform

VivaTrack is a premium frontend fitness tracking, workout planning, progress analytics, and community platform. It is built as a modern SaaS-style product demo using React, Vite, TypeScript, Tailwind CSS, React Router, Lucide icons, and Recharts.

## Features

- Professional startup-style landing page
- Fitness SaaS dashboard preview
- Workout programs page
- Dashboard demo page
- Progress analytics with charts
- Community feed with local demo data
- Pricing plans and revenue model section
- About page and contact form UI
- GitHub Pages deployment workflow

## Tech Stack

React, Vite, TypeScript, Tailwind CSS, React Router DOM, Recharts, and Lucide React.

## Local Run Commands

```bash
npm install
npm run dev
```
